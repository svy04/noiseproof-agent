NARA no-native-text OCR route.

- A deterministic plan artifact for future owner-runtime transcript collection.
- A report builder and check-mode command.
- Source-rights and repository commit boundaries that keep transcript text and
  hashes out of the repo.

## Not Implemented

- Transcript collection.
- Source-rights approval completion.
- Reference transcript availability.
- Transcript hash commitment.
- OCR quality scoring.
- CER or WER computation.
- Robust PDF extraction.
- External reviewer feedback.

## Can Claim

- NoiseProof has an owner-runtime transcript collection plan for the
  source-policy no-native-text OCR route.

## Cannot Claim

- A reference transcript exists.
- Transcript collection has happened.
- Source-rights owner approval is recorded.
- OCR output is correct.
- OCR quality has been benchmarked.
- CER or WER can be computed.
- Robust PDF extraction works.
- The product is complete.
