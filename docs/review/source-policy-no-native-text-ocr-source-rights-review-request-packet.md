# Source-policy No-native-text OCR Source Rights Review Request Packet

Phase marker:
`source_policy_no_native_text_ocr_source_rights_review_request_packet_v0`.

This gate records a source-rights review request packet for the preserved NARA
no-native-text OCR route.

## Implemented

- A deterministic request packet artifact for future rights review.
- A report builder and check-mode command.
- Rights-review questions and repository non-commit boundaries that keep
  transcript text and hashes out of the repo.

## Not Implemented

- Request sending.
- Rights response receipt.
- Rights clearance.
- Source-rights approval completion.
- Transcript collection.
- Reference transcript availability.
- Transcript hash commitment.
- OCR quality scoring.
- CER or WER computation.
- Robust PDF extraction.
- External reviewer feedback.

## Can Claim

- NoiseProof has a source-rights review request packet for the source-policy
  no-native-text OCR route.

## Cannot Claim

- Rights are cleared.
- A request was sent.
- A response was received.
- Source-rights owner approval is recorded.
- A reference transcript exists.
- Transcript collection has happened.
- OCR output is correct.
- OCR quality has been benchmarked.
- CER or WER can be computed.
- Robust PDF extraction works.
- The product is complete.
