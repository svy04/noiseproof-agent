## Short Korean Version

안녕하세요. 저는 고객의 지저분한 데이터 환경에서 근거 없는 AI 답변이 통과하지 못하게 만드는 RAG/Agent 시스템에 관심이 있습니다.

NoiseProof Agent는 시장 문서, CSV, HTML, 메모처럼 형태가 다른 자료를 입력으로 받아 문서 프로파일링, parser boundary, chunk strategy 비교, lexical retrieval, Evidence Ledger Preview, Noise Gate Preview, claim-bounded report preview까지 작게 연결한 포트폴리오 프로젝트입니다.

이 프로젝트는 트레이딩 봇이 아닙니다. buy/sell 추천, 목표가, 수익률 예측을 만들지 않고, 대신 "무엇을 근거로 말할 수 있는가", "어떤 출처가 충돌하는가", "어떤 주장은 아직 막아야 하는가"를 서비스 구조로 보여주는 데 집중했습니다.

Forward Deployed Engineer 포지션에 먼저 맞춘 이유는, 이 프로젝트가 고객의 애매한 데이터 문제를 작은 시스템으로 정의하고 구현하며 운영 표면까지 남기는 방식과 가깝기 때문입니다. 장기적으로는 Product Engineer 수준의 인프라와 제품 책임까지 확장하고 싶습니다.

## Short English Version

I am interested in RAG/Agent systems that prevent unsupported AI answers from passing in messy customer data environments.

NoiseProof Agent is a small portfolio service for noisy market intelligence. It profiles messy inputs, compares parser and chunking boundaries, runs lexical retrieval, builds Evidence Ledger previews, applies a Noise Gate, and only then formats claim-bounded report previews.

It is not a trading bot. It does not provide buy/sell recommendations, target prices, return predictions, or financial advice.

The project is FDE-first because it demonstrates customer-like problem framing, service implementation, operational logging, failure records, and technical decision documentation.
